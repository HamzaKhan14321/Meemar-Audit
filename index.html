Website / Google Search / Blog / Linktree

Conversion hub, SEO asset, project information center, and inquiry-routing system for serious buyers.

550 clicks 10.3K impressions 5.4% CTR Avg. position 7

Data basis: Google Search Console screenshots, last 3 months; website, Linktree, project pages, and WhatsApp routing review.

Verified Data

Google Search Console confirms 550 clicks, 10.3K impressions, 5.4% CTR, and average position 7 over the last 3 months. The website, project pages, Linktree, and WhatsApp are the main buyer-routing assets.

Business Meaning

Search demand already exists. The website should not behave like a brochure, it should convert search traffic into project-page visits, WhatsApp inquiries, brochure requests, and site-visit interest.

Good
  • Official website gives Memaar a controlled destination outside social media.
  • Search Console proves organic demand is present.
  • Project pages can become high-intent landing pages.
  • WhatsApp fits local real estate buyer behavior.
Weak
  • Blog system is not strong enough for the search opportunity.
  • Project pages need clearer payment-plan, location, and inquiry CTAs.
  • Linktree can dilute intent if too many low-priority links are shown.
  • Website clicks are not yet connected to lead/source reporting.
Main Risk

Traffic can be wasted if visitors cannot quickly find project details, payment plans, construction status, WhatsApp, or site-visit action.

How I Will ActWebsite Execution System
  • Build a monthly keyword calendar using Search Console query data, identify the exact phrases driving the current 10.3K impressions and write each blog to answer one of those buyer queries with project-specific depth, not generic real estate content.
  • Link every blog post directly to its most commercially relevant project page via an in-body link and a WhatsApp CTA at the article close, no blog should end without pointing the reader toward a specific next action.
  • Rewrite project page structure to surface payment-plan figures, location context, construction stage, and a WhatsApp button above the fold on mobile so a first-time visitor finds the answer to "is this worth my time" within 10 seconds.
  • Cut the Linktree to 3 visible actions maximum: project information, payment plan, and WhatsApp, every link beyond these three reduces the probability the buyer clicks any of them.
  • Run a monthly Search Console review: track CTR and position changes per query, flag improvements from published blogs, and identify new buyer-intent searches to address the following month so the content calendar stays demand-led.

Facebook

Largest reach engine, local awareness builder, retargeting pool, and premium paid creative testing ground.

4.1M views 1.6M viewers 31K engagement High confidence

Verified Data

Last 90 days: 4.1M views, 1.6M viewers, 31K engagement, 4.9K reactions, 454 comments/replies, 266 shares, approximately 98% non-follower views, and −357 net audience change.

Business Meaning

Facebook is doing the hardest top-of-funnel job, getting attention. The issue is that this attention is not visibly tied to WhatsApp inquiries, project-page clicks, or confirmed buyer actions.

Good
  • Strongest verified reach platform in the ecosystem.
  • Useful non-follower discovery at scale.
  • Best base for retargeting audiences.
  • Strong local buyer potential if routed correctly.
Weak
  • Reach is not mapped to leads or inquiries.
  • Net audience change is negative despite large reach.
  • Some content may be too awareness-heavy, not buyer-intent focused.
Main Risk

High views can create a false sense of performance if they stay passive. Without source tracking, Memaar cannot know which Facebook content creates real buyer action.

How I Will ActFacebook Execution System
  • Operate Facebook as a two-machine system, organic posts build the warm retargeting pool, paid creatives convert it, every piece of content gets one of these two jobs assigned before it is written, never both at once.
  • Build 4 monthly ad creatives using a scripted format: a 1.5-second visual scroll-stop hook using real construction footage, a proof moment showing project scale or delivery quality, one CTA routing directly to WhatsApp, no multi-step link chains.
  • Retarget the 1.6M past viewers and post-engagers first before any cold-traffic spend, this audience already encountered Memaar and costs significantly less to convert than a cold audience who has never seen the brand.
  • Organize organic posts into five defined content lanes: construction progress, payment-plan breakdown, delivery milestone, buyer FAQ, and third-party proof, so the page tells a structured buyer story instead of an unpredictable stream of uploads.
  • Review performance weekly rather than monthly: 3-second view rate on organic videos, link-click rate on ads, and cost-per-result on paid campaigns, pause any creative underperforming after 3 days of data rather than waiting for budget to drain.

How I Will ActInstagram Execution System
  • Rebuild the profile top-to-bottom as a conversion funnel: bio written around one project advantage and one action only, link pointing to a 3-option maximum Linktree, and the top pinned reel answering the buyer question "what is this project and how do I inquire" in under 30 seconds, no music or caption reading required.

  • Measure save rate and DM intent as the primary Instagram quality signals: 200 saves from 1,000 views is commercially stronger than 10,000 views with no action depth, and reel saves indicate the content earned enough trust to be stored for later.

How I Will ActTikTok Execution System
  • Use TikTok Developments as the hook-testing machine, film every core clip with at least two different opening seconds, measure which version holds watch time past the 3-second mark, and push the winning variation to Facebook and Instagram the following week so the best hooks lead the wider distribution.

  • Score TikTok Developments performance by 3-second hold rate, comment intent volume, and profile-visit-to-follow rate, not raw view counts, because a clip that sends 50 people to the profile with buying questions outperforms one that generates 50,000 passive scrolls.
